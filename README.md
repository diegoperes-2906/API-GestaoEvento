## 📄 Descrição do entregável

Sistema CRUD completo para gerenciamento de eventos e participantes. Desenvolvido em **Java** com uso da IDE **Eclipse**, o sistema permite cadastrar, listar, atualizar e excluir tanto eventos quanto participantes, além de gerenciar inscrições.

---

## 💻 Sobre o projeto

O **Gestão de Eventos** foi criado para permitir o controle eficiente de eventos como palestras, workshops e seminários.  
O projeto consiste em classes Java organizadas em pacotes, utilizando conceitos de orientação a objetos e persistência de dados com listas em banco de dados.

---

## ⚙️ Funcionalidades

### 📅 Eventos
- [x] Criar evento (nome, descrição, data, local, vagas)
- [x] Listar eventos disponíveis
- [x] Atualizar dados do evento
- [x] Excluir evento

### 👤 Participantes
- [x] Cadastro de participante (nome, e-mail, telefone)
- [x] Inscrição em evento (com verificação de vagas)
- [x] Cancelamento de inscrição
- [x] Listar participantes de um evento

## 🚀 Como executar o projeto

1. Clone este repositório ou baixe o .zip
2. Abra o Eclipse
3. Vá em: `File > Import > Existing Projects into Workspace`
4. Selecione o diretório do projeto
5. Execute a classe principal (`Main.java`) para iniciar o sistema
